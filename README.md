# 📚 NSDA Level-6 Written Exam Preparation

A structured study guide and topic repository for **NSDA Level-6 Written Examination**.
This repository helps learners prepare for the written exam by organizing important topics from **Mathematics & Statistics, Programming, AI & Machine Learning, and AR/VR / Immersive Technology**.

---

# 🎯 Objective

The main objective of this repository is to:

* Organize important exam topics
* Provide a clear preparation guideline
* Help students revise concepts quickly
* Focus on exam-oriented study

---

# 📂 Subject Areas

## 1️⃣ Mathematics & Statistics

Important topics:

* Mean, Median, Mode
* Variance & Standard Deviation
* Probability Theory
* Bayes Theorem
* Linear Algebra (Matrix, Vector)
* Eigenvalues & Eigenvectors
* Gradient & Partial Derivatives
* Optimization Basics

---

## 2️⃣ Programming

Important topics:

* Python Basics
* Data Types & Variables
* Control Statements (if, loop)
* Functions
* List, Tuple, Dictionary
* File Handling
* Object Oriented Programming
* Basic Algorithms

## 3️⃣ AI & Machine Learning

Important topics:

* Supervised Learning
* Unsupervised Learning
* Regression Algorithms
* Classification Algorithms
* Naive Bayes

## 4️⃣ AR / VR / Immersive Technology
